\doxysubsection{Detailed Description}
Contains the dispatching logic for Cu\+Py \& Num\+Py arrays based on the active \mbox{\hyperlink{namespaceTask}{Task}} Environment. 

This file contains the dispatching logic for Cu\+Py \& Num\+Py arrays based on the active \mbox{\hyperlink{namespaceTask}{Task}} Environment. This is used to determine which array type to use for a given task. The manual data movement functions {\ttfamily copy} and {\ttfamily clone\+\_\+here} are defined here.

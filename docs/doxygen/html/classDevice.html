ResourcePool_t& Device::get_reserved_pool ( )

Returns the currently reserved resources on the device.

This starts at max and decreases as resources are reserved. This represents the resources currently in use by the tasks. This is necessary to determine if tasks can be scheduled without oversubscription or OOM errors.

    Class InnerTask

    ClassList > InnerTask

    The C++ "Mirror" of Parla's Python Tasks This class is used to create a C++ representation of a Parla Task All scheduling logic should be handled by these after creation until launched by the Python callback.

    • #include <runtime.hpp>

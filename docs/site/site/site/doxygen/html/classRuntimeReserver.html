Detailed Description

RuntimeReserver phase of the scheduler.

Reserves all 'non-persistent resources. This plans task execution on the device set. Here all 'non-persistent resources that have a lifetime equal to the task body are reserved and are not directly shared between tasks. At the moment this is only the VCUS/Threads a task uses. This phase submits the task to the launcher.

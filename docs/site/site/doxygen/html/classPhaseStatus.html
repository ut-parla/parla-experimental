Detailed Description

template<typename S>
class PhaseStatus< S >

Records metrics that track phase execution (e.g. success, failure, etc.)

Template Parameters
SThe enum class that defines the states to track.

Used to record counts of phase execution for tracing and debugging. For example, the number of successful mappings per call.

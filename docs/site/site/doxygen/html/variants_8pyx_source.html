96  A decorator to declare that this function has specialized variants for specific architectures.

98  The default can just be `raise NotImplementedError()` in cases where no default implementation is possible.
99  To provide a specialized variant use the `variant` member of the main function:
100  .. testsetup::
101  from parla.function_decorators import *
102  >>> @specialize
103  ... def f():
104  ... raise NotImplementedError()
105  >>> @f.variant(architecture)
106  ... def f_gpu():
107  ... ...

110  Each architecture can only be used once on a given function.

\doxysubsection{Detailed Description}
A worker is a thread that executes tasks. 

The worker is a C++ object that is created by the Python runtime. It is responsible for executing tasks assigned to it by the scheduler. The worker sleeps until it receives a task from the scheduler. When it recieves a task, it wakes up and acquires the GIL.

\doxysubsection{Detailed Description}
Python \mbox{\hyperlink{classparla_1_1cython_1_1tasks_1_1Task}{Task}} interface. 

This class is used to represent a task in the task graph.

A task is a unit of work that can be executed asynchronously. Tasks are created by calling the spawn decorator on a python code block. Tasks are scheduled for execution as soon as they are created.

The task class is a wrapper around a C++ task object. The C++ task object is created when the task is spawned and is destroyed when all references to the task are gone. The python interface stores the Python function task body and passes all metadata (mapping and precedence constraints) to the C++ runtime on creations.

39  New in Numpy version 1.20.0.

114  """Converts numpy/cupy ndarray to Parla array without creating additional copy.
115 
116  Args:
117  ```ndarray(s)```, or ```ndarray(s)``` in ```dict/list/tuple/set``` (could be nested).
118  Its structure will be kept.

121  the same number of Parla array that matches the inputs.
122 
123  Example:
124  a = numpy.array([1,2])
125  b = [cupy.array([3,4]), cupy.array([3,4])]
126 
127  a, b = asarray_batch(a, b) # a and b are now parla array

129  def get_parray(object): # recursively process Sequence or Dictionary
130  if isinstance(object, (numpy.ndarray, cupy.ndarray)):
131  return asarray(object)
132  elif isinstance(object, PArray):
133  return object
134  elif isinstance(object, dict):
135  accumulator = {}
